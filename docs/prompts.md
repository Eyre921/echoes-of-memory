# AI提示词设计文档

## 概述

本项目使用精心设计的提示词来指导AI的行为，确保其能够以温暖、有同理心的方式与用户交互，并有效地帮助用户记录和整理人生回忆。

## 核心提示词

### 破冰提示词

用于开启每日对话，建立友好关系：

```
你好！今天过得怎么样？我注意到你最近提到了一些关于[话题]的事情，让我很好奇。你愿意和我分享更多关于那个时期的细节吗？比如，那时候你最喜欢做什么？有什么特别难忘的人或事吗？
```

### 深度探索提示词

用于挖掘用户分享内容的更多细节：

```
你刚才提到的[具体内容]真的很有趣！我特别想了解更多关于[具体事件/人物]的细节。你能描述一下当时的情景吗？比如周围的环境、你当时的感受，或者还有谁在场？这些细节对我来说都很珍贵。
```

### 情感共鸣提示词

用于表达对用户情感的理解和共鸣：

```
我能感受到你对[话题]的[情感]。这确实是一个让人[相关情感]的经历。你愿意告诉我更多关于这段经历对你的影响吗？
```

### 记忆整理提示词

用于帮助用户整理和组织记忆：

```
根据我们之前的对话，我整理了关于[主题]的几个要点：
1. [要点1]
2. [要点2]
3. [要点3]

你觉得还有其他需要补充的吗？或者你想对某个要点进行更详细的描述？
```

## 回忆录生成提示词

### 结构化生成提示词

用于生成结构化的回忆录内容：

```
请根据用户的所有记忆数据，按照以下结构生成回忆录章节：

## 第一章：童年时光
- 重点描述用户的童年经历、重要人物和关键事件
- 保持温暖、怀旧的语调
- 字数控制在800-1000字

## 第二章：求学岁月
- 描述用户的求学经历、重要成就和难忘回忆
- 突出成长和学习的过程
- 字数控制在800-1000字

## 第三章：初入社会
- 描述用户初入职场的经历、挑战和收获
- 强调适应和成长的过程
- 字数控制在800-1000字

请开始生成第一章的内容。
```

### 风格化生成提示词

用于生成不同风格的回忆录内容：

```
请将以下内容转换为[风格]风格：

原文：
[用户原始内容]

要求：
1. 保持原意不变
2. 采用[风格]的表达方式
3. 增强文字的感染力和可读性
```

可选风格包括：
- 平实朴素
- 文艺抒情
- 风趣幽默
- 哲理思辨

## 最佳实践

1. **个性化** - 根据用户的表达习惯和偏好调整提示词
2. **情境化** - 结合当前对话情境使用合适的提示词
3. **渐进式** - 从简单问题逐步深入到复杂话题
4. **情感化** - 关注用户情感变化，适时给予情感支持

## 持续优化

1. 定期分析用户反馈，优化提示词效果
2. 根据用户画像调整提示词策略
3. 收集高质量对话样本，完善提示词库
